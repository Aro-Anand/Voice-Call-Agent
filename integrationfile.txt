Integration with Existing Website
To integrate with an existing website, you have two options:

Embed the Widget Script: Add this script tag to your existing website:

html<script src="https://your-deployment-domain.com/widget/call-widget.js"></script>

Iframe Integration: If you prefer isolation:

html<iframe src="https://your-deployment-domain.com" style="display:none" id="callAssistantFrame"></iframe>
<script>
    // Simple control script
    function openCallAssistant() {
        const frame = document.getElementById('callAssistantFrame');
        frame.style.display = 'block';
        frame.style.position = 'fixed';
        frame.style.bottom = '0';
        frame.style.right = '0';
        frame.style.width = '450px';
        frame.style.height = '600px';
        frame.style.border = 'none';
        frame.style.zIndex = '9999';
    }
    
    // Add a button to your site
    const btn = document.createElement('button');
    btn.textContent = 'Talk to Agent';
    btn.onclick = openCallAssistant;
    document.body.appendChild(btn);
</script>
